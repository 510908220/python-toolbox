# python-toolbox

## 使用

- 需要使用`jupyer`,如果没安装的话可以执行命令`pip install jupyter`进行安装
- 执行命令`git clone https://github.com/510908220/python-toolbox.git`
- `cd books`,执行`jupyter notebook`,然后浏览器上去查看操作.
